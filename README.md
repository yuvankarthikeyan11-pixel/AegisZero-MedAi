# AI Pneumonia Detection

This project detects pneumonia from chest X-rays using Deep Learning.

## Features
✔ Chest X-ray validator  
✔ Pneumonia detection  
✔ DenseNet121 model  
✔ Confidence scoring  

## Tech Used
- TensorFlow
- OpenCV
- DenseNet121

## How to Run
1. Install requirements
2. Run final_predict.py
